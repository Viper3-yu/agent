# Perplexity

> AI 搜索引擎 / 研究 Agent。实时搜索互联网并综合回答，附带来源引用。

## 基本信息

| 项目 | 详情 |
|------|------|
| 厂商 | Perplexity AI |
| 官网 | https://perplexity.ai |
| 发布日期 | 2022-12 |
| 定价 | Pro $20/月, Enterprise $40/月 |
| 目标用户 | 研究人员、知识工作者、开发者 |
| 底层模型 | GPT-4o, Claude Sonnet, 自研模型（混合路由） |

## 核心能力

1. 实时网络搜索
2. 自动来源引用
3. 深度研究模式（Deep Research）
4. 多步搜索推理

## 工作流

```
用户输入问题
  ├─ Perplexity 解析意图 → 生成搜索查询
  ├─ 实时搜索互联网 → 抓取多个来源
  ├─ 综合分析 → 生成结构化回答
  ├─ 附带来源引用 → 用户可追溯
  └─ 追问 / Deep Research → 深入探索
```

## 技术架构

- **搜索**: 自有爬虫 + 实时索引
- **模型**: 多模型混合路由（GPT-4o, Claude, 自研）
- **特色**: RAG（Retrieval-Augmented Generation）架构，来源可追溯
- **模式**: Quick Search / Deep Research / Focus（学术/写作/代码）

## 与同类对比

| 特性 | Perplexity | ChatGPT | Google Search |
|------|-----------|---------|---------------|
| 优势 | 来源引用、实时搜索、结构化回答 | 通用对话能力强 | 网页覆盖广 |
| 劣势 | 生成式幻觉风险 | 无实时搜索（默认） | 无综合回答 |
