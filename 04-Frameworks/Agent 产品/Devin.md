# Devin

> Cognition 推出的"AI 软件工程师"。自主完成从需求理解到代码提交的全流程。

## 基本信息

| 项目 | 详情 |
|------|------|
| 厂商 | Cognition |
| 官网 | https://devin.ai |
| 发布日期 | 2024-03 |
| 定价 | $500/月（Team Plan） |
| 目标用户 | 工程团队、需要自动化开发任务的企业 |
| 底层模型 | 未公开（推测多模型混合） |

## 核心能力

1. 全自主：从理解需求到 PR 提交
2. 自带开发环境（终端、编辑器、浏览器）
3. 持续学习项目代码库

## 适用场景

- 独立完成小型 feature
- Bug 修复
- 代码迁移

## 工作流

```
用户提交任务（自然语言描述）
  ├─ Devin 理解需求 → 拆解子任务
  ├─ 启动隔离开发环境（沙箱）
  ├─ 浏览器搜索文档 / 阅读代码库
  ├─ 编写代码 → 运行测试 → 修复问题
  ├─ 创建 PR → 等待用户 review
  └─ 根据反馈迭代
```

## 技术架构

- **环境**: 沙箱化开发环境（终端 + 编辑器 + 浏览器）
- **模型**: 未公开，支持长任务规划和执行
- **特色**: 端到端自主，无需人工干预（中途可介入）
- **集成**: GitHub / GitLab PR 流程
