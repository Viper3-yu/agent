## 四、后续建议

**等老大在本地解除沙箱**（或从外网拉资料）后，可重做联网核查：

1. **模型笔记**（22 篇）—— 拉各厂商 release page，确认日期/版本/上下文窗口/定价，把骨架填实
2. **框架笔记**（10 篇）—— 拉 spec URL 的最新 release tag，确认 API/字段
3. 验证通过后 → 升 mature + 移除 meta/stale + 移除 ⚠️ 警告行
4. 验证失败（如版本已过期）→ 改内容 + 留 reviewed
