# 书单与课程

> 想精读 / 已开读 / 已读完的资源清单。每项有**进度**和**为什么收藏**两列——不堆链接。
> 配套 RSS：[[../RSS订阅清单 使用说明|RSS 订阅清单]]（Tier 2 有 4 个相关源）。

## 维护原则

1. **每项必填 2 个理由**："为什么收藏" + "建议时机"（入门 / 进阶 / 专题）
2. **进度用 checkbox**：`[ ]` 未开读 / `[~]` 在读 / `[x]` 读完
3. **每月只开 1 本**（A 线节奏）
4. **读完一本必发 vault 复盘笔记**（强迫消化）
