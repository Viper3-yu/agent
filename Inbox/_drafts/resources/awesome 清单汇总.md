# awesome 清单汇总

> 收集 GitHub 上**质量经过社区筛选**的 awesome-xxx 仓库清单，作为发现新资源的入口。
> 配套：[[论文清单]] / [[书单与课程]] / [[../RSS订阅清单 使用说明|RSS 订阅清单]]。

## 维护原则

1. **只收"被大量 star + 仍在维护"**的清单（star < 1k 不收）
2. **每个清单点开扫一遍再收**，不抄名字
3. **季度回顾**：检查 star 增长 + 最后 commit，停滞的降级
4. **替代**：`awesome-AI-feeds` 见 [[../RSS订阅清单-AI-Agent.opml#工具/镜像（备用）|OPML 工具镜像]]——awesome 仓库的二级链接
