# 论文清单

> 按主题分组的 AI/Agent 论文索引。**只列已经读过/已建笔记的论文**，不堆 arXiv 链接（那太容易过期）。
> 配套 RSS 源：[[../RSS订阅清单 使用说明|RSS 订阅清单]]（arXiv cs.AI/CL/MA/LG + HF Daily Papers）。

## 维护原则

1. **宁缺勿滥**：每篇论文要看过摘要 + 至少 1 篇 vault 笔记引用它才收
2. **不复制 arXiv**：arxiv ID 不直接列在这，只在 vault 笔记里出现
3. **主题分组粗一点**：5-7 个一级主题足够，再细就建子清单
4. **季度回顾**：每季度清一次"读了但 3 个月没引用"的论文
