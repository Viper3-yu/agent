# 常见失败模式

> Prompt 设计中的常见陷阱

## 幻觉诱导

当 prompt 的表述暗示了某个不存在的事实时，模型可能会顺着这个暗示编造信息。

**示例**：问"爱因斯坦在哪一年获得诺贝尔化学奖？"——实际上他获得的是物理学奖，模型可能编造一个年份。

## 过度约束

给模型施加过多限制条件，导致模型无法灵活处理边界情况，或在约束之间产生冲突。

## 指令冲突

prompt 中存在相互矛盾的指令，导致模型行为不确定或偏向某一指令。

**示例**：同时要求"尽量简洁"和"提供详细的解释"。

## 格式错误

指定的输出格式与模型能力不匹配，或格式描述不够清晰，导致输出格式不稳定。
