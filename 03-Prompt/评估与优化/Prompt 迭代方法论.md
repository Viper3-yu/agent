# Prompt 迭代方法论

> 系统化地优化 prompt 效果的方法论

## 迭代循环

1. **定义目标**：明确期望的输出质量和行为
2. **编写初版**：基于最佳实践编写初始 prompt
3. **测试评估**：使用多样化输入测试 prompt 效果
4. **分析差距**：对比期望输出与实际输出的差异
5. **优化调整**：基于分析结果修改 prompt
6. 重复以上循环直到达到满意效果

## 评估维度

### 准确性
- 输出内容是否事实正确
- 推理过程是否逻辑自洽
- 是否包含幻觉或编造信息

### 一致性
- 相似输入是否产生一致的输出
- 输出格式是否稳定
- 多次运行结果是否可复现

### 安全性
- 是否能抵御 prompt 注入攻击
- 是否泄露系统 prompt 内容
- 是否生成有害或不当内容

## A/B测试

通过对比不同版本 prompt 的表现来选择最优方案：
- 控制变量，每次只修改一个因素
- 使用相同的测试用例集
- 量化评估指标（准确率、格式符合率等）
